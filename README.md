# ES6+ TODO LIST 학습 프로젝트

JavaScript **ES6+ 문법을 많이 사용**해서 만든 TODO LIST 예제입니다.

## 실행 방법

브라우저에서 `index.html`을 열면 바로 실행됩니다.

## 학습 포인트 (ES6+)

- `const` / `let`
- Arrow Function
- Template Literal
- Destructuring
- Spread / Rest
- Class + Private Field (`#`)
- `Map`, `Set`
- `Object.freeze`
- Optional Chaining (`?.`)
- Nullish Coalescing (`??`)
- `Array.map/filter/reduce/find/findIndex/some/every`
- Generator (`function*`)
- `async/await` + `Promise`

## 기능

- 할 일 추가 / 삭제 / 완료 토글
- 전체/진행중/완료 필터
- 우선순위(높음/중간/낮음) 선택
- 태그 입력 및 중복 제거
- 완료 항목 일괄 삭제
- 남은 할 일 카운트 + 완료율 표시
- `localStorage` 자동 저장/복원
